Creating dashboards

To understand your data better, you must create a dashboard that displays different charts, statistics, and tables. This can give you more insight into your data.

Before you begin

You must have completed the following tasks:
  • Been assigned the role of an Administrator of Grafana.
  • Installed the IBM速 RMF for z/OS Grafana App plugin. See Installing the plugin on Grafana.
  • Added a RMF data source in Grafana. See Creating RMF data sources.
  • Understood the query language of the RMF data source. See Query languages.

Procedure

  1. Navigate to the left-side menu, and then click Dashboards.
  2. Click New, and then select New Dashboard.
  3. Click + Add visualization to add visualization to your data.
    The Edit panel is displayed.
  4. Enter the panel title in the Title field.

    Optionally you can also provide a description for the panel that you are creating.

  5. Select one of the following from the visualization types drop-down menu:

    Option Description
    Bar Chart For more information, refer to the Bar charts section in the Grafana documentation.
    Report for IBM RMF for z/OS This panel visualization enables you to create graphs in table format.
  6. Select the existing data sources from the Data source drop-down list.
  7. Click the Query tab, and then enter a query in the query language of the RMF data source.
    Note: You can click + Query to add multiple queries.
  8. Click the Refresh dashboard icon to query the RMF data source.
    Grafana provides you with a preview of your query results along with the corresponding visualization.
  9. Optional: Click Apply to view your changes applied to the dashboard.
  10. Click Save and perform the following steps to save the dashboard:
    1. Enter a name for the dashboard in the Dashboard name field.
    2. Select a location to save your dashboard from the Folder list.
      Note: The default location for the Folder is set as General.
    3. Click Save to save the new dashboard.

Results

You have created the dashboard.

What to do next

You can perform the following tasks after you create the dashboard:
  • Add more panels to the dashboard by clicking Add > Visualization from the dashboard header.

  • Configure the panel options based on your requirements. For more information refer to the Configure panel options topic in the Grafana documentation.
  • Customize the dashboard settings, For more information refer to the Modify dashboard settings topic in the Grafana documentation.

  • Perform various tasks such as creating a folder, moving dashboards into folders, deleting multiple dashboards, and navigating to a folder page where you can assign folder and dashboard permissions. For more information, refer to the Manage dashboards topic in the Grafana documentation.
