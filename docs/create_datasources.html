Creating RMF data sources

To access RMF Monitor III metrics in Grafana, you need to connect to the Distributed Data Servers by adding an RMF data source.

Before you begin

You must have completed the following tasks:
  • Been assigned the role of an Administrator of Grafana.
  • Installed the IBM® RMF for z/OS Grafana App plugin. See Installing the plugin on Grafana.
  • Copied the hostname and port number of the DDS (v2) to be used during the creation of the data source process.

Procedure

  1. Go to Apps > IBM RMF
    Note: The navigation of the Grafana user interface can differ based on the Grafana version that is currently installed.
  2. Click Add RMF Data Source.
  3. Enter a name for the data source in the Name field.
  4. Optional: Click the icon to make the added data source the default one.
    Note: When you create new panels, the default data source is preselected.
  5. Enter the port number of the DDS in the Port field.
  6. Enter the hostname of the DDS in the Server field.
  7. Optional: Click the icon to enable the secure connection
    Note: When you enable SSL, the Verify the server’s certificate chain/hostname option is also enabled. You can click to turn off the server certificate chain or hostname verification.
  8. Enter the credentials of the DDS in the Username and Password fields.
  9. Click Save & test.
    The Data source is working message is displayed.

Results

You have added the RMF data source.

What to do next

You can create dashboards to visualize RMF Monitor III data and metrics. See Creating dashboards.
