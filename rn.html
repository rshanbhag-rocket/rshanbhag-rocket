Release notes

This article covers the new features, system requirements, and known problems of the IBM® RMF for z/OS Grafana App plugin.

What's new

2023-09-29

The first release of the IBM® RMF for z/OS Grafana App plugin is in the GitHub repository: https://github.com/IBM/RMF.

Requirements

You must have the following software to use the IBM® RMF for z/OS Grafana App plugin:
  • z/OS Resource Measurement Facility 3.1
  • Distributed Data Server v2
  • Grafana v9.2.5 or later

Known issues

There are no known issues in this release.
